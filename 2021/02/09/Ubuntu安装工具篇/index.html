1.安装VMware Tools

1.1 选择安装/重新安装VMware tools

1.2 VMware会自动下载最新的安装包(默认会在以下目录：/media/<你的用户名>/VMware Tools)

1.3 进入安装包目录

1.4 解压安装包(指令：tar -zxvf VMwareTools-10.2.5-8068393.tar.gz)

如果报以下错误：chmod: changing permissions of ‘VMwareTools-10.2.5-8068393.tar.gz’: Read-only file system

使用以下指令，将压缩包解压到可写目录即可：tar -zxvf VMwareTools-10.2.5-8068393.tar.gz -C /home/jackou/Desktop

1.5 进入vmware-tools-distrib然后运行./vmware-install.pl 按默认安装一直回车到底即可。、

1.6 安装成功
